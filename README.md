Data Analytics Practice

Peranan Data dalam Keputusan Bisnis

Repository ini dibuat untuk memenuhi tugas mata kuliah **Data Infrastructure and Analytics**.

📌 Deskripsi
Project ini membahas peranan data dalam pengambilan keputusan bisnis serta praktik dasar analisis data menggunakan Python melalui Google Colab.

📂 Isi Repository
* `Untitled0.ipynb` → Notebook praktik analisis data
* `artikel-peranan-data.docx` → Artikel teori dan praktik
* `screenshots/` → Dokumentasi praktik (jika ada)

🛠 Tools yang Digunakan
* Python
* Pandas
* Google Colab
* GitHub

📊 Praktik yang Dilakukan
1. Menjalankan Python dasar
2. Import library pandas
3. Memuat dataset publik
4. Menampilkan data awal (head)
5. Melihat struktur data (info)

🎯 Tujuan
Memahami konsep data, data-driven business, serta praktik dasar eksplorasi data menggunakan Python.
